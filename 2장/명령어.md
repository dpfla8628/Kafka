토픽 생성
```
./kafka-topics.sh --create --bootstrap-server ip:port --topic hello1
// 파티션 개수 , 복제 개수, 토픽의 데이터 유지 기간 설정
./kafka-topics.sh --create --bootstrap-server ip:port --partitions 3 --replication-factor 1 --config retention .ms=1728000000 --topic hello2
```
토픽 리스트 조회
```
./kafka-topics.sh --bootstrap-server ip:port --list
====
hello1
hello2
```
상세 조회
```
./kafka-topics.sh --bootstrap-server ip:port --describe --topic hello1
```
